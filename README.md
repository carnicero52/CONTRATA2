# ContrataFácil - Versión Simple

Sistema de recepción de currículums que guarda los datos localmente.

## Características

- ✅ Registro de empresas
- ✅ Formulario público para candidatos
- ✅ Panel de administración
- ✅ Gestión de estados de candidatos
- ✅ Exportar a CSV
- ✅ Carga de CV (PDF, DOC)

## Tecnologías

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- localStorage (datos guardados en el navegador)

## Importante

**Los datos se guardan en localStorage del navegador.** Esto significa:
- Solo visibles en el mismo navegador
- Se pierden si se limpia el navegador
- No se comparten entre dispositivos

Para una versión con base de datos en la nube, ver la versión Pro.

## Despliegue

```bash
npm install
npm run build
npm start
```
